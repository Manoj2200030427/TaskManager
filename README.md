# 🗂️ Task Management System (MERN Stack)

A full-stack Task Management web application built using the **MERN stack (MongoDB, Express, React, Node.js)**. This project helps users efficiently create, update, track, and manage daily tasks with authentication and role-based access.

---

## 🚀 Features

- 👤 User Authentication (Register/Login using JWT)
- 📝 Create, Update, Delete Tasks
- 📊 Task Status Tracking (Pending / In Progress / Completed)
- 🔍 Filter & Sort Tasks
- 🔐 Protected Routes
- 👨‍💼 Role-based access (Admin/User - optional)
- 📱 Responsive UI (Mobile + Desktop)

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Axios
- React Router DOM
- Tailwind CSS / CSS

### Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt.js
